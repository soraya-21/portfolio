## Mon Portfolio

Je suis ravie de vous présenter mon portfolio codé from scratch en HTML et intégrant (évidemment) du CSS et du Javascript.
L'objectif est de le mettre à jour au fur et à mesure et de garder une trace des différents projets sur lesquels je travaille.

Il a été déployé avec Netlify: https://app.netlify.com/
